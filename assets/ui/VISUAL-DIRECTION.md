# THE MATTMACHINE — Visual Upgrade Direction

## Phase 2 Style Goals

- darker graphite surfaces
- restrained neon hierarchy
- CRT scanlines
- smoked glass panels
- arcade hardware energy
- reactive glow motion
- cyberpunk dashboard feel

## UI Hierarchy

### Brightest
- transport
- active pads
- recording states

### Medium
- waveform editor
- selected pad
- visualizer

### Darkest
- inactive pads
- background panels
- utility controls

## Pad State Colors

- cyan = looping/sync
- pink = glitch/reverse/fx
- green = solo/active
- yellow = armed/chop/edit
- dark gray = muted/inactive

## Motion System

Add:
- pad pulse on hit
- transport shimmer
- waveform breathing
- visualizer strip
- subtle idle movement
- glow decay

## Asset Direction

Avoid:
- flat gradients
- toy-like rainbow blocks
- generic DAW sliders

Use:
- tiny LEDs
- brushed metal
- dark acrylic
- subtle grime
- chrome edges
- icon overlays

## Layout Direction

TOP
- transport
- visualizer strip

CENTER
- waveform editor

BOTTOM
- performance pads

DRAWER
- B-SIDE FX
- advanced controls
