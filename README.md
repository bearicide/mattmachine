# THE MATTMACHINE

Mobile-first MATTBEAR sampler prototype.

## Current build

- Single-file `index.html`
- Neon MATTMACHINE visual direction
- 8 individual pads
- Touch-first mobile controls
- Built-in synth drum / glitch / arcade packs
- Phone microphone recording
- Long-press pad to arm recording
- REC / STOP captures audio to that pad
- Volume and pitch controls

## Live URL

After GitHub Pages is enabled:

https://bearicide.github.io/mattmachine/

## Notes

This is the starter build. MIDI, quantized loop launching, uploaded sample packs,
PWA install support, and Drive/shared upload flow are next-phase features.
